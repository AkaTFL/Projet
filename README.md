# Mon Projet Web

Un projet simple avec HTML, CSS et une CI de tests.

## Structure du projet

```
├── index.html      # Page principale
├── css/
│   └── style.css   # Styles CSS
├── js/
│   └── app.js      # JavaScript
└── .github/
    └── workflows/
        └── ci.yml  # Configuration CI
```

## Fonctionnalités

- ✅ Page d'accueil responsive
- ✅ Styles CSS modernes
- ✅ Tests automatisés via GitHub Actions

## Lancer le projet

Ouvrez simplement `index.html` dans votre navigateur.

## Tests

Les tests sont exécutés automatiquement via GitHub Actions à chaque push et pull request.

```bash
npm install
npm test
```
